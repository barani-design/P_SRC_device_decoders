How to parse with Python parser in CMD line (Power Shell/CMD)

###############################

1.Download files from git.
2.Go to downloaded file location and pick folder with parser what you want.
3.Open your terminal (Win+R and type "cmd" then hit enter).
4.Navigate terminal to your downloaded folder, for example cd C:Desktop\Your\Path\To\File
5.In terminal use python eg. -> python MeteoALTIM_periodic_payload.py yourPayloadString
6.Hit enter and enjoy parsed payload
