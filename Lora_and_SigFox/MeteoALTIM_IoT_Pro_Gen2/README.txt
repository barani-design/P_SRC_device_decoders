How to decode with Python decoder in CMD line (Power Shell/CMD)

###############################

- Download files from git.
- Go to downloaded file location and pick folder with parser what you want.
- Open your terminal (Win+R and type "cmd" then hit enter).
- Navigate terminal to your downloaded folder, for example cd C:Desktop\Your\Path\To\File
- In terminal use python eg. -> python MeteoALTIM_periodic_payload.py yourPayloadString
- Hit enter and enjoy parsed payload
